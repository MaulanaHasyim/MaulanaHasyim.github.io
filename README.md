# Maulana Hasyim — Portfolio

Website portofolio pribadi. Data & Business Analyst.

## Cara hosting di GitHub Pages

1. Buat repository baru bernama **`MaulanaHasyim.github.io`** (harus sama persis dengan username GitHub).
2. Upload **`index.html`**, **`profile.jpg`**, dan **`README.md`** ke root repo.
3. Buka **Settings → Pages**, pilih branch `main` dan folder `/ (root)`, klik **Save**.
4. Tunggu 1–2 menit. Situs live di **https://maulanahasyim.github.io**

## Struktur file
- `index.html` — halaman utama (HTML + CSS + JS dalam satu file)
- `profile.jpg` — foto profil

## Update konten
Semua teks ada di dalam `index.html`. Edit langsung lalu commit ulang.
